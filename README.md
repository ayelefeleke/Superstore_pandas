Title: Superstore Data Analysis using Pandas
 Project Objective:
This project analyzes the Superstore dataset using Python Pandas to extract meaningful business insights such as sales trends, profit distribution, customer behavior, and regional performance.
            Dataset
The dataset contains retail sales data including:
  Order details (Order ID, Order Date, Ship Date)
  Customer information
 Product categories and sub-categories 
 Sales, Profit, Discount
  Region and Segment

 Tools & Libraries Used
- Python
- Pandas
- Matplotlib
- Seaborn
 Google Colab
 tasks on this project
1. Data Loading & Exploration
 Loaded dataset using Pandas
 Checked structure, data types, and summary statistics
 2. Data Cleaning
 Handled missing values
 Converted date columns to datetime format
 3. Feature Engineering
 Extracted Month and Year from Order Date
Created Profit Margin column

 4. Data Analysis
Sales by Category
Profit by Region
 Top customers
Monthly sales trends

5. Pivot Tables
Sales by Category and Region
Sales by Segment over time
Profit by Sub-Category
 6. Data Visualization
Line chart for sales trend
 Bar charts for category and region analysis
Pie chart for segment distribution

  # The Insights are
 Technology category generates the highest profit
 Some regions perform better than others
 Sales do not always equal profit due to discounts
 Monthly sales show seasonal patterns
